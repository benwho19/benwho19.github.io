Housing regression

View code
In this project, we're working with a housing dataset. It contains data such as Avg Area Income, Avg Area House Age, Price, and Area Population. Our goal is to perform a data analysis on it in order to figure out which features are most closely associated with a house's price. Afterwards, we will train a linear regression model to try to predict a house's price based on its other features.
Technologies used: Python, Pandas, Numpy, Matplotlib, Seaborn, Scikit-learn.
Here's a quick snapshot of what the data looks like:
After getting a brief look at the data by using various Pandas methods (such as .head, .describe, etc.), we conducted an exploratory data analysis to do a deeper dive into the data. We used Seaborn's jointplot method, which draws a plot of two variables with bivariate and univariate graphs, to look for any relationships between features.
One of the last visualizations we created was a heatmap of these correlations, which is shown here on the right. We see that a feature like 'Avg Area Income' is correlated quite strongly with price, whereas a feature like 'Avg Area Number of Bedrooms' has a weak correlation with price.
After finishing the EDA, we then wanted to build a linear regression model. So we split our data into a training set and test set.
After first training our model to make predictions on price from other features, we make our test predictions and then evaluate the model's performance. Here is a scatterplot of the model's residuals to the right. We see that it has performed quite well.
We also evaluated the model's coefficients and see that they are normally distributed, which tells us this model was a good choice for this data.
